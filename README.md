# Vietnam PDP8 Analysis

This project analyzes Vietnam’s Power Development Plan 8 (PDP8), focusing on planned electricity capacity changes between 2030 and 2050.

## Files
- `vietnam_pdp8_analysis.Rmd` – full R code
- `vietnam_pdp8_analysis.pdf` – final report
- `vietnam_pdp8_data.csv` – cleaned dataset

## Key Findings
- Total installed capacity increases strongly by 2050.
- Coal declines significantly in the long term.
- Solar and wind become major power sources.
- Vietnam moves toward a lower-carbon electricity mix.

## Tools Used
R, tidyverse, ggplot2
